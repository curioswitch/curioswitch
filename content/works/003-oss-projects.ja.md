## CurioSwitch のオープンソースプロジェクト

CurioSwitch では、日々の開発で培った技術基盤をオープンソースとして公開しています。\
私たちの開発姿勢である 「小さなチームでも高品質を実現する仕組みづくり」 を、多くの開発者に届けたいという思いから生まれた取り組みです。

# 🚀CurioStack — フルスタック開発を加速する基盤

CurioStack は、Web アプリケーションを素早く構築するための フルスタック開発キット です。
プロジェクト構成、API・フロントエンドのテンプレート、CI/CD、GCP 設定、セキュリティまでを網羅し、
開発者がすぐに実装に集中できる環境を提供しています。

- Go / Connect / Protocol Buffers による高速・型安全な API
- 安全なデフォルト設計（TLS / 相互認証など）
- GitHub Actionsによる自動デプロイ
- モノリポ構成でスケールしやすいプロジェクト管理
- MIT License で自由に利用・再利用可能

> GitHub: https://github.com/curioswitch/go-curiostack
> GitHub: https://github.com/anuraaga

# 私たちの OSS に込めた思い

CurioSwitch のオープンソースは、単なるコード公開ではなく、\
「よりよい開発体験を広げる」ための共同プロジェクトです。

- 新しいプロダクトをすぐに立ち上げたいチーム
- インフラ構築に時間をかけたくない開発者
- 安全性とスピードの両方を求めるプロジェクト
- 少人数でエンタープライズ品質を目指すスタートアップ
  こうした開発現場で役立つことを願い、継続的に改善しながら公開を続けています。
